# Transmit — FTP/SFTP/S3 Client From Panic With Beautiful Interface

## About Transmit

Transmit provides an elegant FTP, SFTP, WebDAV, S3-compatible cloud storage client for macOS, crafted by Panic Inc. Transmit combines a gorgeous dual-pane interface, lightning-fast transfers, server-to-server copy (without local download), sync bookmarks (smart folder of favorite connections), and cloud service integration (Backblaze B2, Google Drive, Microsoft Azure, IBM Cloud Object Storage). Transmit enables users to upload websites, manage server files, sync backups, and transfer between clouds effortlessly. Transmit is trusted by web developers, designers, system administrators, and creative professionals who appreciate both form and function.

## Deep Dive — Elegant File Transfer for macOS

Protocols — FTP, FTPS (explicit/implicit), SFTP, WebDAV, WebDAVS (HTTPS), S3 (Amazon, Wasabi, Backblaze B2, Google Cloud Storage, Azure, IBM Cloud, MinIO, DigitalOcean Spaces)  
Interface — dual-pane browser (local + remote), tabbed connections, favorites bar, spring-loaded navigation  
Performance — multi-threaded transfers (up to 20 concurrent connections), optimized for high-latency SFTP, resume interrupted transfers, checksum validation  
Sync tools — Transmit Sync (two-way synchronization like rsync), download/upload queue, scheduled sync via Automator  
Integration — Finder extension (mount Transmit connections as drives), Terminal integration (open current path in Terminal), panicsync (bookmark cloud sync)  
Unique features — server-to-server copy (transfer between two remote servers directly), panic sync (sync bookmarks across Macs), diskmode (browse external drives)

## Quick Reference for Transmit

| Operation | Transmit Action |
|-----------|-----------------|
| New connection | Cmd+N |
| Connect to favorite | Click favorite in sidebar |
| Upload (local → remote) | Drag file from left to right pane |
| Download (remote → local) | Drag file from right to left pane |
| Server-to-server copy | Drag between two remote tabs |
| Sync folder | Tools → Sync → Select direction → Preview → Sync |
| Show transfer queue | Cmd+Shift+Q |
| Add to favorites | Drag connection icon to Favorites sidebar |
| Mount as drive | File → Mount Connection (Finder extension) |
